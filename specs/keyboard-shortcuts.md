# Keyboard shortcuts

Quick reference for all available keyboard shortcuts.

## Shortcuts

| Key                        | What it does                                     |
| -------------------------- | ------------------------------------------------ |
| **W**                      | Enter waypoint placement mode                    |
| **P**                      | Enter POI placement mode                         |
| **O**                      | Open the orbit template                          |
| **G**                      | Open the grid survey template                    |
| **F**                      | Open the facade scan template                    |
| **Z**                      | Open the pencil path template                    |
| **B**                      | Enter obstacle drawing mode                      |
| **Escape**                 | Cancel the current action or deselect everything |
| **Delete** / **Backspace** | Remove the selected waypoint, POI, or obstacle   |
| **Ctrl+A** / **Cmd+A**     | Select all waypoints                             |

## Good to know

- Shortcuts only work when you're not typing in a text field.
- You can also access all these tools from the toolbar buttons on the map.
